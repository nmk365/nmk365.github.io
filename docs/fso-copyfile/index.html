FSO - ファイルをコピーする | CopyFileメソッド

ファイルをコピーするには CopyFile メソッドを使う。

fso.CopyFile( Source, Destination, [OverWriteFiles = True] )

Source にはコピー元ファイルのパスを指定する。
ワイルドカードも使用可能。

Destination にはコピー先のパスを指定する。

OverWriteFiles にはファイルを上書きするかどうかを指定する。
True を指定した場合、既存のファイルは上書きされる。
False を指定した場合、上書きされない。
既定値は True 。

コピー先が読み取り専用の場合、 OverWriteFiles の指定に関わらず処理は失敗する。
